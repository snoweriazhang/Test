Setup Version Control: Git

  1. Download latest version of Git for OS X 10.7
  2. Create a folder in which to place your git repos. mkdir -p ~/repos/git/working cd ~/repos/git/working
  3. Create a Github account and verify your account email address.
  4. Configure your global git config with your account information.
  5. Generate SSH keys for Github

  6. Create a repo (with Github)

    1. Create a new repo on Github (with the with README option).
    2. Clone the repo. (Make sure you are in your git working directory.) pwd # this should be ~/repos/git/working git clone git@github.com:<your username>/<your repo>.git cd <your repo>
    3. Make changes and push pwd # this should be ~/repos/git/working <do stuff ...> git add <files> git commit -m <message> git push # fetch and merge changes from remote repo (remember to COMMIT before doing this) git pull # pull down changes from remote repo (remember to COMMIT before doing this)

Setup Local Webserver: MAMP

  1. Download MAMP

  2. Setup Virtual Hosts

    • Only once, you need to uncomment Include ../httpd-vhosts.conf
    • For each web app, choose a domain name to add to /etc/hosts
      • #dobby
127.0.0.1           test.local                  # dobby's first web app
    • For each web app, add a new VirtualHost and DocumentRoot declaration to Include ../httpd-vhosts.conf
      example 
      <VirtualHost *:80>
    ServerAdmin admin@localhost
    DocumentRoot "C:/repos/Test/web"
    ServerName test.local
    ErrorLog "logs/test.local-error.log"
    CustomLog "logs/test.local-access.log" combined
    <Directory "C:/repos/Test/web">
      Order Deny,Allow
      Allow from all
    </Directory>
</VirtualHost>
